# eDCP-Chinese-translation-patch
    本项目为麦当劳 eCDP 的中文化补丁，基于游戏文本提取及 LLM 翻译制作。  
    当前版本已基本达到可游玩状态，绝大多数游戏文本能够正常显示。但现阶段无论是翻译内容还是文本排版均不作任何质量保证。大量文本尚未经过系统性的校对、润色和界面适配，因此玩家可能会遇到翻译生硬、术语不统一、未翻译内容、换行异常、文本溢出、间距问题以及其他显示错误。  本版本更适合作为一个具有潜力的早期验证成果，而非完成度较高的正式本地化版本。其主要目标是降低语言门槛，并为后续完善和社区贡献提供基础。  本仓库仅提供补丁数据，不包含任何受版权保护的游戏资源。
    要游玩本中文化版本，请您自行准备合法来源的原版 eCDP ROM，并使用 DeltaPatcher 应用本项目提供的补丁。请务必提前备份原始文件。补丁内置 ROM 校验机制，仅支持指定版本的未经修改的原版 ROM，不支持任何已修改、已打补丁或其他改版文件。
